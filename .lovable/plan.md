

# TEU Alumni Network 구현 계획

## 개요
흑백 미니멀 디자인의 TEU 동문 네트워크 웹앱. Lovable Cloud를 백엔드로 사용하여 DB, 인증, 스토리지, 실시간 채팅을 구현합니다. 뉴스 기능은 이후 단계에서 추가합니다.

---

## 1단계: 데이터베이스 & 인증 기반

### 데이터베이스 테이블
- **alumni_profiles**: 이름, 닉네임, 기수, 직장/회사, 직함, 관심사, 기여, 기대, SNS, 이메일, 프로필 사진 URL, 접속코드
- **access_codes**: 16자리 코드(xxxx-xxxx-xxxx-xxxx), 동문 연결 여부, 생성일
- **messages**: 발신자, 수신자, 내용, 읽음 여부, 타임스탬프 (실시간 채팅용)
- **site_content**: 페이지별 텍스트/이미지 편집을 위한 CMS 테이블

### 인증 방식
- 동문: 16자리 접속 코드로 로그인 → 본인 프로필만 수정 가능
- 관리자: ID `info@te.university` / PW `teuteu1919`로 별도 로그인

---

## 2단계: 핵심 페이지 구현 (한국어 UI)

### 홈페이지
- TEU Alumni 메인 문구, 미니멀 흑백 히어로 섹션
- 관리자가 편집 가능한 텍스트/이미지 영역

### 동문 디렉토리
- 기수별 필터링 (TEU 1~6, TEU ART 1~3, TEU MED 1~5)
- 카드 레이아웃: 사진, 이름, 닉네임, 기수, 직장, 직함, 관심사
- 카드 클릭 시 상세 프로필 모달 (SNS, 이메일, 기여/기대 등 포함 + DM 보내기 버튼)

### 내 프로필 페이지
- 프로필 폼 (전체 필드), 사진 업로드
- 저장 시 디렉토리에 공개

### 네트워크 그래프
- Obsidian 스타일 인터랙티브 그래프 (흑백/회색)
- 관심사, 기여, 기대 텍스트 기반 유사도로 엣지 생성
- 기수 필터, 유사도 임계값 슬라이더, 이름 검색
- **이름 입력 시**: 현재 임계값 기준 관련 동문 카드가 기수 아래에 표시 (% 일치율)
- **관련 동문 외 이름/카드 클릭 시**: 디렉토리와 동일한 상세 프로필 표시
- **관련 동문 노드 클릭 시**: 사이드 패널에 미니 프로필 + 관련 동문 목록

---

## 3단계: 실시간 DM 채팅

- 프로필 상세에서 "DM 보내기" 버튼
- Supabase Realtime 기반 1:1 실시간 채팅
- 받은 메시지 목록, 읽음/안읽음 표시
- 채팅 인터페이스 (흑백 디자인)

---

## 4단계: 관리자 모드

### 관리자 대시보드 (로그인 필수)
- 전체 동문 프로필 조회 및 수정
- 접속 코드 랜덤 생성기 (xxxx-xxxx-xxxx-xxxx, 알파벳+숫자 조합)
- 동문 이름 + 접속 코드 목록 표시
- 전체 동문 프로필 엑셀 다운로드 버튼
- 홈페이지 텍스트/이미지 편집 기능 (블로그 스타일 CMS)

---

## 5단계: 디자인 & UX

- **색상**: 흑백만 사용 (검정, 흰색, 회색 톤)
- **타이포그래피**: 깔끔한 산세리프 한글 폰트
- **카드 UI**: 미세한 그림자, 얇은 보더, 일관된 여백
- **반응형**: 모바일/데스크탑 대응
- **네비게이션**: 홈 / 디렉토리 / 네트워크 그래프 / 내 프로필
